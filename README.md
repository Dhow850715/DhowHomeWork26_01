# SnapStitch - 圖片快速合併工具

這是一個專業的照片快速合併工具，支援多張圖片拖曳合併、佈局調整、壓縮預覽以及簡易圖片編輯。

**注意：本專案僅供個人測試與技術展示用途。**

### 主要功能
- **多圖合併**：支援不限張數的照片合併。
- **佈局選擇**：提供網格 (Grid)、橫向 (Row)、縱向 (Col) 三種合併模式。
- **解析度控制**：支援原始解析度或自定義壓縮比例。
- **容量預估**：即時顯示原始與壓縮後的檔案大小。
- **即時編輯**：支援對單張圖片增加文字、方框與圓形標記。
- **隱私安全**：採用純客戶端處理技術，所有操作均在瀏覽器本地完成，照片不會上傳到任何伺服器。

### 使用技術
- React 19
- Tailwind CSS (CDN)
- Canvas API (圖片處理)
